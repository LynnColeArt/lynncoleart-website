CSS Opacity Test

This page tests the opacity effect for gallery backgrounds.

If working correctly, you should see a very faint (11% opacity) background image behind the content.

Gallery Item with Background

This is a test gallery item. The background image should be visible at 11% opacity.

The text should be fully opaque and readable.

Another Gallery Item

This uses a different background image, also at 11% opacity.

How it works (old school method):

  • Separate div for background with its own opacity
  • Background div is positioned absolutely
  • Content div is positioned relatively with higher z-index
  • No inheritance issues, no pseudo-element complications
